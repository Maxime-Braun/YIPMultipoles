How to use this tool

                
Welcome to the Multipole Calculator! This tool helps you determine allowed magnetic or electric multipoles for atoms in a crystal based on symmetry.

            

                
Step 1: Select a Magnetic Space Group

                    
Start by identifying the Magnetic Space Group (MSG) or conventional space group of your crystal structure. You can:

                
    
                    
  • Type the BNS number (e.g., "1.1" or "62.448") in the input box.
    • 
                    
  • Or select the group directly from the dropdown list.
    • 
                

            

            
            

                
Step 2: Choose a Wyckoff Position

                
Select the Wyckoff position where your magnetic atom is located. This is crucial because the local site symmetry determines which magnetic moments are allowed.

                
Note: The calculation will start automatically once you select a position.

            

            
            

                
Step 3: Set Maximum Rank

                
Decide the highest multipole rank you want to calculate:

                
    
                    
  • Rank 1: Magnetic Dipole (Vector)
    • 
                    
  • Rank 2:  Magnetic Quadrupole
    • 
                    
  • Rank 3: Magnetic Octupole
    • 
                

                
Higher ranks reveal more complex magnetic distributions allowed by the crystal symmetry.

            

            
            

                
Step 4: Interpret Results

                    
Step 4: Interpret Results

                
    
                    
  • Site Symmetry Operations: Lists the specific rotations and mirrors that leave the atomic site invariant.
    • 
                    
  • Allowed Multipoles: Shows the mathematical form of the allowed tensor. Variables (e.g., x, z) represent independent degrees of freedom.
    • 
                    
  • Atom Ordering: Visualizes the relationship between the representative atom (Atom 1) and other atoms in the same orbit.
                        
      
                                
    • Red (UP): Magnetic/electric multipole is parallel to Atom 1.
      • 
                                
    • Blue (DOWN): Magnetic/electric multipole is antiparallel to Atom 1.
      • 
                                
    • Group badge: Shows `Group N` for groups with more than one member. Badge color indicates group relations: green = all members parallel, red = contains antiparallel members, neutral = non-collinear.
      • 
                                
    • Ordering summary: The panel shows an overall ordering label derived from group relations (e.g., `Ferroically Order`, `Anti-Ferroically Order`, `Mixed / Non-collinear ordering`, or `NCL order`).
      • 
                                
    • Type toggle: Use the Type radio control to switch between Magnetic and Electric multipoles. Magnetic mode applies the axial/time-odd parity factor; Electric mode treats tensors as polar/time-even.
      • 
                                
    • Sublattice translation: For Rank-1 dipoles the tool attempts to find a pure-translation symmetry mapping the `Same` sublattice to the `Opposite` sublattice and displays the fractional translation `t` when found.
      • 
                        
    
                    
    • 
                

                
Tip: Toggle the "Voigt Notation" switch to simplify the display of higher-rank tensors (Rank 2+).
